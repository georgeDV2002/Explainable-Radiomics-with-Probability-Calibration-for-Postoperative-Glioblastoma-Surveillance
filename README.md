# Explainable-Radiomics-with-Probability-Calibration-for-Postoperative-Glioblastoma-Surveillance
Code for Explainable Radiomics with Probability Calibration for Postoperative Glioblastoma Surveillance. Includes preprocessing, radiomics extraction, feature selection, LightGBM training, Optuna tuning, calibration, and SHAP explainability. Uses the MU-Glioma-Post TCIA dataset.
